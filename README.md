## 🧰 Chapter Structure

Every chapter follows a consistent 5-part structure:

1. **Intuition** — Plain-English motivation, no jargon
2. **Theory** — Mathematical formulation with LaTeX-rendered equations
3. **Algorithm** — Step-by-step pseudocode
4. **From-Scratch Python** — Complete, runnable implementation using only the standard library
5. **Exercises** — Practice problems and extensions

## 🚀 Quick Start

### Option 1 — Google Colab (no setup required)

Click any **Open in Colab** badge in the table above to run a chapter directly in your browser.

### Option 2 — Run locally

```bash
# Clone the repo
git clone https://github.com/zia207/ML-python.git
cd ML-python

# Install Quarto (https://quarto.org/docs/get-started/)
# Then render the full site:
quarto render

# Or run a single notebook:
jupyter notebook Colab_Notebook/chapter-02-linear-regression.ipynb
```

**Requirements:** Python 3.9+ with `jupyter` installed (`pip install jupyter`). No other ML packages needed — all algorithms use only the Python standard library.

## 🛠️ Philosophy

> *"The best way to understand an algorithm is to implement it from scratch."*

This tutorial uses **only Python's standard library** for all core algorithm implementations — no `sklearn`, no `torch`, no `tensorflow`, no `numpy`. Just `math`, `random`, and built-in data structures.

This forces genuine understanding. You can't call `.fit()` and move on — you have to know what's happening inside.
