## 🎯 How to Use This Tutorial

### Learning Paths

**🟢 Beginner Path** (Chapters 00 → 01 → 02 → 03 → 04 → 05 → 15)
Start here if you're new to machine learning or Python.

**🟡 Intermediate Path** (Chapters 00–05 → 06 → 07 → 08 → 12 → 13 → 15)
For those comfortable with Python who want to understand classical ML deeply.

**🔴 Advanced Path** (Full sequence, Chapters 00–15)
Complete end-to-end study including deep learning and RL.

## 🧰 Philosophy

Each chapter follows this structure:

1. **Intuition** — Plain-English explanation of the idea
2. **Theory** — Mathematical formulation (notation explained for beginners)
3. **Algorithm** — Step-by-step pseudocode
4. **From-Scratch Python** — Pure Python implementation (no scikit-learn, no TensorFlow)
5. **Worked Example** — A real dataset or toy problem
6. **Exercises** — Practice problems to reinforce learning
